新版item布局组件页面 校验页面1 校验页面2 树的加载 输入框只读/编辑状态切换

公共JS方法使用说明：

添加jquery插件:linkCss();

    注:统一页面中所有链接的样式

    例: CSS添加: <span class="link-css" id="link_address" > 地址链接 </span>

        JS调用: $('#link_address').linkCss();

loading(type,msg); 加载中的提示遮罩

    注:在提交请求事件之前调用loading('open','显示的信息'),请求成功后调用loading('close);

    例: $('#btn').on(click,function(){
            loading('open','正在加载中,请稍候...');
            $.ajax({
                success : function(){
                    loading('close');
                }
            });
        });

crossAddTab(tab_title,tab_url,tab_id); 打开新的标签页面

    注: tab_title : 新标签页的title,(尽量保证title为唯一)
        tab_url : 新标签页面的url,
        tab_id : 新标签的ID,(作为切换,关闭刷新页面的标示,必须确保为唯一)

    例: $('#btn').on(click,function(){
            crossAddTab('实有人口新增',basePath + '/syrkGl/syrkAdd','syrkAdd_tab_id');
        });

comboAutoComplete(combobox_id,url); combobox自动完成数据填入

    注: combobox_id : combox的ID
        url : 下拉框字典数据地址
        当下框的值只有一条时,调用该方法,自动完成填写,组件生成后调用

    例: $(function(){
            comboAutoComplete('combox_id',basePath + '/syrkGl/syrkAdd');
        });

打开远程页面(URL)加载的弹窗的方法

    例:  openUrlForm({
             id: 'dlg_id',       //自定义ID,确保唯一性,防止重复添加DOM
             url: '2.html',      //需要放入弹框的页面URL
             title: '表单提交',   //模态框标题
             width: 800,         //模态框宽度
             height: 600,        //模态框高度
             cache:true,         //是否缓存
        });

注意事项：

1.所有easyui的input组件,请添加val类以提高批量处理的效率

    例: <input type="text" class="easyui-combobox val">
