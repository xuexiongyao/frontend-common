/api/xlpc/queryPageList

巡逻盘查被盘查人员分页查询

/api/xlpc/queryPageList post

post

post /api/xlpc/queryPageList

  • Request
  • Response

Query Parameters

  • ry_id: (string)

    被盘查人员id

    Example:

    37a088dcef0f426ca8d7334ef7f44e83
  • bpcr_xm: (string)

    被盘查人员姓名

    Example:

    210203195410034804
  • bpcr_xb: (string)

    被盘查人员性别代码

    Example:

    01
  • bpcr_mz: (string)

    被盘查人员民族代码

    Example:

    01
  • page: (integer)

    当前页数，从1开始

    Example:

    1
  • rownum: (integer)

    每页条数

    Example:

    10
  • rows: (integer)

    当前第几条

    Example:

    10

HTTP status code 200
