  • Request
  • Response

Query Parameters

  • rzbt: (string)

    日志标题

    Example:

  • rznr: (string)

    日志内容

    日志分类

  • ydqx: (string)

    是否公开阅读，0否，1是

    Example:

    测试日志内容

HTTP status code 200

Body

Type: application/json
