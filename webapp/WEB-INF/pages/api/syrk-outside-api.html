/api/syrk/ryxxApply get

人员信息复用，先复用警综库（即实有人口）,如果没有，再复用户籍系统（常住人口）；可通过有证人员（cyzjdm+zjhm）或者无证人员（xm+csrq+csrqend）两种方式复用

get

get /api/syrk/ryxxApply

  • Request
  • Response

Query Parameters

  • cyzjdm: (string)

    成员证件代码，即证件类型，111表示身份证

  • zjhm: (string)

    证件号码

  • xm: (string)

    姓名

  • csrq: (string)

    出生日期，yyyy-MM-dd格式

  • csrqend: (string)

    出生日期结束日期，无证人员的出生日期是个时间段，yyyy-MM-dd格式

  • gxdw: (string)

    管辖单位，过滤居住地时使用，有值得时候只查询这个单位下的居住地，无值得时候返回这个实有人口的所有居住地

  • system: (one of JZXT, BJXT, SJXT)

    复用的系统，如果有值，就只复用这个系统的数据，如果没有，则按顺序复用
